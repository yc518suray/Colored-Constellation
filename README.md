# Colored-Constellation
A program to visualize constellation points in communications.

## Usage
Download the m file, and run it.

The function `show_color_constellation` is not in the form of a single function file. You can do this for yourself.

Code to simulate simple doubly-selective channel is provided. The modulation symbols are placed on the DD domain.
